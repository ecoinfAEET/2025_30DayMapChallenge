30DayMapChallenge 2025

Reto colaborativo, noviembre, 2025
https://30daymapchallenge.com/

Instrucciones para participar:

  1. Selecciona el da en la tabla que ms te guste y apntate! Djanos tu usuario de GitHub para que podamos darte acceso al repo!

  2. En la rama correspondiente al da, edita el RMarkdown con el cdigo para que renderice el mapa que hayas hecho (ecuerda incluir todos los datos en la carpeta del da, dentro de 2025_30DayMapChallenge_files, no te olvides de hacer knit)!

  3. Cada da, aproximadamente a las 18:00h, integraremos el mapa del da a la rama main. Publicaremos los resultados en nuestro Bluesky!
