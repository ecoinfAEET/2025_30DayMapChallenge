# 30DayMapChallenge 2025

Reto del mes de noviembre, 2025. <https://30daymapchallenge.com/>

Instrucciones para participar:

1.  Selecciona un día en la [tabla](https://tinyurl.com/2ba78kmk) y apúntate al día que quieras! Déjanos tu usuario de GitHub para que podamos darte acceso al repo!

2.  En la rama correspondiente al día, edita el RMarkdown con el código para que renderice el mapa que hayas hecho (no te olvides de hacer *knit*)!

3.  Cada día, aproximadamente a las 18:00h, integraremos el mapa del día a la rama *main*. ¡Publicaremos los resultados en nuestro Bluesky!
